ZfmFoundation
=
This _Zend Framework 2_ module provides your application with integration of [Foundation 3][F3],  
a responsive front-end CSS/SCSS framework.

### Framework Specs
  * __Grid System__: 
    *  12-column
    *  Nests  
    *  Offsets  
    *  Centered Columns  
    *  Mobile Grid  
    *  Block Grids
  * __Media Queries__

[F3]: <http://foundation.zurb.com>
